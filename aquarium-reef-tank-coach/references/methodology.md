# Aquarium Troubleshooting Methodology

## Rule 1: context beats numbers

A parameter value matters only with tank age, livestock, trend, source water, and recent changes.

## Rule 2: stability first

Many crashes worsen because users change lighting, dosing, filtration, water changes, and medication at the same time.

## Diagnostic categories

- Cycling / ammonia / nitrite
- Oxygen / flow / temperature
- Disease / parasites
- Compatibility / aggression
- Nutrient imbalance / algae
- Light stress
- Contamination
- Acclimation / recent additions
