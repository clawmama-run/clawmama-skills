# General Parameter Ranges

These are generic references, not species-specific guarantees.

| Parameter | Freshwater common | Reef common | Notes |
|---|---|---|---|
| Ammonia | 0 | 0 | Any detectable ammonia can be urgent. |
| Nitrite | 0 | 0 | Especially dangerous in new tanks. |
| Nitrate | varies | often <5-20 ppm | Depends on livestock/corals/plants. |
| Temperature | species-specific | ~24-26°C | Stability matters. |
| Salinity | n/a | ~1.025-1.026 sg | Confirm with calibrated tool. |
| Alkalinity | n/a | ~7-11 dKH | Avoid rapid swings. |
